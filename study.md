# 랭체인(LangChain)
각 기능을 유연하게 연결할 수 있는 특징을 가짐
예를 들어, 문서 검색, 데이터 처리, 요약, 번역과 같은 여러 작업을 마치 레고 블럭처럼 조립하여 원하는 기능 구현 가능
## 랭체인 코어(langchain-core)
랭체인의 중심 역할 패키지
- 대규모 언어 모델, 데이터 벡터 저장소, 검색기와 같은 중요한 기능들을 정의하는 기본 구조가 포함됨
    - 벡터 저장소란? 데이터를 숫자 형태(벡터)로 저장하는 방식
## 랭그래프(LangGraph)
그래프 기반 모델링을 도와주는 패키지
여러 작업을 동시에 처리하거나 특정 조건에 따라 작업을 분기하는 복잡한 애플리케이션을 설계할 수 있음
## 랭서브(LangServe)
랭체인의 체인을 REST API로 간편하게 배포할 수 있도록 하는 패키지
## 랭스미스(LangSmith)
LLM 어플리케이션을 디버깅, 테스트, 평가, 모니터링할 수 있는 개발자 플랫폼
애플리케이션이 안정적이고 신뢰성 있게 작동하는지 확인할 수 있는 중요한 도구

## 랭체인의 주요 장점
모듈성
- 모든 기능을 독립적인 모듈로 제공
- 단독 사용과 조합 사용 모두 가능하여 앱 구조를 효율적으로 설계할 수 있음
통합 용이성
- 다양한 외부 시스템과 쉽게 연동 가능
커뮤니티와 지원
- 활성화된 커뮤니티에서 다양한 예제와 문서 제공
- 지속적인 업데이트

## 랭체인의 주요 활용 사례
1. 검색 증강 생성(RAG)과 질의응답 시스템
2. 구조화된 출력
    - JSON, XML 등 구조화된 출력 지원
3. 챗봇 구축
    - 챗봇 개발의 복잡한 부분을 단순화하여 쉽고 빠른 개발이 가능
4. 도구 사용 및 에이전트
    - 대규모 언어 뫃모델이 스스로 판단하여 데이터를 처리하고 도구를 사용하는 개념인 에이전트를 랭체인으로 구현 가능
    - 예를 들어, 날씨 API를 호출해 실시간 데이터를 가져오거나 데이터베이스에서 정보를 조회하는 등의 작업을 자동화 가능
